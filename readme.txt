== Description ==

CategorySeo will be able to set to the string to be used in the title element to the registration screen and editing screen categories.

== Installation ==

Installation using "Add New Plugin"
1. From your Admin UI (Dashboard), go to Plugins. Click the "Add New" button.
2. Search for plugin, or click the "Upload" button, and then select the plugin you want to install.
3. Click the "Install Now" button.

Manual installation
1. Upload the "category-seo" folder to the "/wp-content/plugins/" directory.
